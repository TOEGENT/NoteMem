NoteMem — это Android-приложение для удобного хранения и **повторения конспектов по методу интервальных повторений**. Оно совмещает текст, изображения и аудио, чтобы создавать богатые по содержанию записи и эффективно их запоминать.

---

## 🚀 Основные возможности

* ✍️ **Текстовые конспекты** — быстрый ввод и редактирование текста.
* 📸 **Фото** — добавление изображений через камеру или галерею.
* 🎙 **Аудио** — запись и хранение голосовых заметок.
* ⏳ **Интервальное повторение** — настройка интервалов (по умолчанию 480, 2880, 8640 минут) для повторения материала.
* 🏷 **Фильтры** — организация конспектов по темам, поиск и сортировка.
* 🎨 **Кастомизация** — выбор цветовой темы, лимита отображаемых элементов, пути для хранения файлов.
* 🔄 **Автоматическая проверка дат** — приложение само определяет, какие записи пора повторить, и помечает их как «учить».

---

## 📂 Архитектура приложения

* `MainScreen` — главный экран со списком конспектов, поиском, фильтрами и кнопками перехода.
* `EditorScreen` — редактор отдельного конспекта: текст, фото, аудио.
* `SettingsScreen` — настройки приложения (фильтры, интервалы, цветовая схема, лимит строк и др.).
* `CameraClick` — работа с камерой.
* `Mp3Screen` — запись и добавление аудиофайлов.
* `FilterSettings` / `IntervalsScreen` — управление фильтрами и интервалами.

Все данные (тексты, даты, изображения, аудио) сохраняются локально в `.pickle`-файлах.

## 📊 Как работает интервальное повторение

Каждому конспекту задаётся дата создания и текущий интервал (минуты до следующего показа).
После повторения интервал увеличивается по заданной прогрессии. Если интервал достиг максимального значения — запись остаётся в памяти как выученная.
